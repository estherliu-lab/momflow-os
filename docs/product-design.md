# Product Design

## Positioning

MomFlow is a gentle AI workspace for fragmented creators. It is not a traditional todo app and not just an AI copywriting box.

中文定位：给碎片时间创作者的温柔 AI 工作台。  
English positioning: A gentle AI workspace for fragmented creators.

## Core Memory Points

1. **Hold one idea first**: save a tiny thought before it disappears.
2. **Resume after interruption**: keep the next step, not the whole perfect plan.
3. **Create from fragments**: turn notes into posts, scripts, and storyboards.

## Interface Mood

- Warm cream paper background
- Soft brown typography
- Sage, oat, blush, and golden action tiles
- Otto the otter as a friendly companion, not a loud mascot
- Dense enough for daily use, gentle enough for tired users

## Installation Retention

The product should actively guide users to add the PWA to the home screen:

- Show a clear install card on the home screen.
- Explain iPhone and Android steps in plain language.
- Use the browser install prompt when available.
- Keep the icon emotionally recognizable.

## Desktop Companion Behavior

A PWA can support:

- Otto as the home-screen icon
- Notification reminders after permission
- App badge counts for unfinished content
- Long-press shortcuts on supported systems
- In-app Otto states and animations

A PWA cannot make Otto freely move around the phone desktop. That would require a native app or platform widget.
