# Contributing

MomFlow should stay warm, lightweight, and easy to customize.

## Product principles

- Do not turn the home screen into a complex task manager.
- Keep the three memorable entry points strong: idea capture, interruption recovery, video storyboard.
- Do not pressure users to be productive every day.
- Keep prompts editable and platform templates replaceable.
- Prefer local-first behavior before adding server dependencies.

## Development

```bash
npm install
npm run dev
npm run build
```

Use focused changes. If you add a platform, also add a prompt template and an example output.
